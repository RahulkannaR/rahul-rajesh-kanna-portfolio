Ransomware Attack Detection

Sub Heading

Duration: Sep 2024 - Dec 2024

With the rise of wireless communication, dynamic spectrum management has become crucial‚Äîespecially in Cognitive Radio Networks (CRNs), which improve spectral efficiency. However, their open and adaptive nature makes them vulnerable to attacks such as jamming, Primary User Emulation Attacks (PUEA), and Spectrum Sensing Data Falsification (SSDF). This paper introduces a novel deep learning model, the Leaky Integrated Bidirectional Echo State Network (LIBESN), for effective attack detection in CRNs. LIBESN enhances traditional Echo State Networks by integrating bidirectional data flow and leaky integration, enabling better temporal pattern recognition and long-range dependency learning. Unlike conventional recurrent networks, LIBESN uses fixed reservoir weights, reducing training time while maintaining high accuracy. Experiments on a custom CRN dataset show that LIBESN achieves 94.67% accuracy, 93.33% precision, 95.89% recall, and a 94.59% F1-score‚Äîsurpassing existing models. ROC and precision‚Äìrecall analyses further validate its reliability under real- time and imbalanced conditions. LIBESN proves to be a robust, lightweight, and scalable solution for real-time CRN attack detection, paving the way for intelligent security in spectrum- aware systems.

Prediction & Analysis of Air Quality using ANN & CNN

Python code - Integrated Machine Learning Technique

Duration: Jan 2024 - May 2024

Air quality is a major environmental and health concern, especially in developing countries like India. Tamil Nadu, a state in southern India, faces severe air pollution problems due to various sources such as vehicular emissions, industrial activities, biomass burning, and dust storms. To monitor and forecast air quality, conventional methods such as ground-based stations and satellite observations have limitations in terms of spatial and temporal coverage, cost, and maintenance. Therefore, there is a need for alternative methods that can leverage the advances in machine learning and big data analytics to provide high-resolution and real-time air quality information. In this paper, we propose a novel approach for air quality analysis and prediction in Tamil Nadu using artificial neural networks (ANNs) and convolutional neural networks (CNNs). These are powerful machine learning techniques that can learn complex nonlinear relationships from large and diverse datasets, and can handle missing and noisy data. We use historical data of air pollutant concentrations, meteorological variables, and land use features from various sources to train and test our models. We compare the performance of our models with existing methods and evaluate their robustness and generalizability. We also provide insights into the spatiotemporal patterns and trends of air quality in Tamil Nadu, and identify the key factors and drivers of air pollution. Our approach can be useful for policy makers, researchers, and citizens to understand and improve the air quality situation in Tamil Nadu and other regions with similar characteristics.

Smoke Detector

using Arduino IDE

Nov 2023

In this project, I designed and built a smoke detection system using an Arduino microcontroller to provide a low-cost, real-time fire safety solution. The core of the system is the MQ-series gas sensor (specifically the MQ-2 or MQ-135), which is capable of detecting smoke, as well as gases like methane, carbon monoxide, and LPG. The sensor continuously monitors the air quality and outputs an analog signal proportional to the concentration of smoke particles in the air. The Arduino reads the sensor's analog output and compares it to a predefined safety threshold. If the smoke level exceeds this threshold, the system is programmed to trigger both visual and audible alerts‚Äîa red LED flashes, and a buzzer sounds continuously‚Äîto warn occupants of potential fire hazards. To enhance usability and testing, the project also included: - Calibration logic to adjust sensitivity based on environmental conditions. - Real-time monitoring through the Arduino Serial Monitor for debugging and data observation. - Modular design that allows for easy integration with IoT systems or home automation platforms in the future. This project demonstrates practical skills in embedded systems, sensor interfacing, real-time data processing, and basic safety system design. It highlights how simple hardware components can be combined to build functional prototypes with real-world applications.

Ping Pong Game

using Arduino IDE

Duration: Feb 2023

Single-Player Ping Pong Game Using Arduino Uno and OLED Display This project involved developing a single-player Ping Pong game using an Arduino Uno and a 0.96‚Äù OLED display based on the SSD1306 driver, creating an engaging and interactive mini-game using minimal hardware. The game interface is rendered on the OLED screen, which communicates with the Arduino via the I2C protocol. I2C (Inter-Integrated Circuit) is a two-wire serial communication method that enables efficient and fast data transfer between the microcontroller and peripheral devices like the OLED, ensuring smooth graphics updates without lag. Key features and technical details: - Game Mechanics: The player controls a paddle located at the left edge of the screen, moving it vertically to hit a bouncing ball. The ball rebounds off the paddle and screen edges, simulating basic physics. If the player misses the ball, the game resets or deducts a life point. - Input Control: A potentiometer or push-button setup was used to control the paddle's movement, providing intuitive and responsive interaction. - Graphics Rendering: The SSD1306 OLED library was used to draw game elements like the paddle, ball, and boundary lines. Despite the limited resolution (128x64 pixels), the graphics are sharp and functional due to the monochrome display and optimized rendering functions. - Game Loop: The core game logic was written in Arduino C/C++, including collision detection, ball velocity control, paddle movement, and scorekeeping. The update loop maintains consistent frame rates for a smooth user experience. - Power Efficiency: The OLED display and I2C communication are low-power, making this project suitable for battery-operated or portable builds. This project demonstrates fundamental concepts in embedded programming, game logic development, display interfacing, and real-time user interaction. It reflects how even simple components can be creatively combined to produce interactive and entertaining applications.
